# ProjectPrice Branding

## Canonical Logo Asset

Use [Logos/app_icon_base.jpg](../Logos/app_icon_base.jpg) as the single source of truth for ProjectPrice branding.

## Usage Rules

- Flutter splash/logo asset should be sourced from `apps/flutter_app/assets/images/app_icon_base.jpg`.
- Theme colors should follow the logo palette:
  - Primary navy: `#0E3A78`
  - Accent emerald: `#16A36A`
- Alternate draft concepts are archived under [Logos/archive](../Logos/archive).

## Asset Flow

1. Finalize logo in [Logos/app_icon_base.jpg](../Logos/app_icon_base.jpg).
2. Copy it into app assets at `apps/flutter_app/assets/images/app_icon_base.jpg`.
3. Keep archived concepts in [Logos/archive](../Logos/archive) for reference only.
